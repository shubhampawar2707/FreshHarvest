 FreshHarvest is a **full-stack web application** that enables users to explore and purchase organic groceries with a seamless shopping experience.  
Built with a **React + Bootstrap frontend** and a **Spring Boot + MySQL backend**, it offers secure authentication, product management, and smooth order handling.


User Features
- Browse and search organic products by category.
- Add/remove items from the cart.
- View product details with images and prices.
- Place orders securely.
- Register and log in with authentication.
- Responsive UI for all devices (mobile, tablet, desktop).

 Admin Features
- Add, update, or delete products.
- Manage user orders.
- View product inventory.
- Dashboard for analytics (optional enhancement).

  **Frontend**  React, React Router, Bootstrap, Axios 
  **Backend**  Spring Boot, Hibernate, REST API 
  **Database**  MySQL 
  **Version Control**  Git + GitHub
